>git init [to initialize git]

>git status [to list all untracked files]

>git add -A [to adding all untracked files in staging area]

>git commit -m "commit message"[to commit added files to local repository]
>git clone [to take exact copy from remote repository to local workspace]
>git pull [to take changes from remote repository to local workspace]
>git push [to sent changes from local workspace to remote repository]
>git config --list[to list configaration]
